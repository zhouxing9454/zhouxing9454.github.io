hexo搭建个人博客流程

hexo和github搭建个人博客

1.安装node.js

https://www.cnblogs.com/liuqiyun/p/8133904.html

检查安装是否成功：

  • node -v
  • npm -v

2.安装git

https://www.cnblogs.com/xueweisuoyong/p/11914045.html

检查安装是否成功：

  • git –version

3.下载Hexo

安装命令：npm install hexo-cli -g

检查安装是否成功：

  • hexo -v

4.Github账号注册

https://www.bilibili.com/read/cv5107169/

5.github仓库搭建

先点击+号，再点击new repository。

6.生成ssh Keys

复制这个文件里面的东西

7.本地生成博客内容

新建文件夹用于存放博客

git bash here

8.发布博客到互联网上

生成界面东西全选就好，30天

9.主题配置

https://github.com/fluid-dev/hexo-theme-fluid

10.配置分类和标签

输入以下命令：hexo new page categories

输入以下命令：hexo new page tags

11.发布文章

发布文章时，在文章开头加：

  • hexo clean
  • hexo g
  • hexo s
  • hexo d
